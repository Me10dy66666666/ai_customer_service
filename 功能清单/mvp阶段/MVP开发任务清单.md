# AI智能客服 MVP开发任务清单

根据《AI智能客服系统技术方案设计.md》拆解的MVP（最小可行性产品）开发任务清单。
本清单严格遵循**“知识库搭建 → 多渠道问答 → 用户权限区分 → 咨询记录分析”**的业务闭环，并按优先级进行了排序。

## 一、 MVP开发任务清单

### 1. 核心闭环任务 (优先级：高)
这些任务是系统跑通的基石，必须优先完成。

| 任务ID | 任务名称 | 模块 | 输入 | 输出 | 验收标准 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **H-01** | **数据库初始化** | DB | SQL脚本 | `users`, `knowledge_base`, `consultation_logs` 等表结构 | 数据库表创建成功，字段类型与设计文档一致，无报错。 |
| **H-02** | **用户注册/登录接口** | 后端/User | 用户名, 密码, 手机号 | JWT Token, 用户信息 | 1. 未注册用户可注册。<br>2. 登录成功返回有效JWT。<br>3. 密码加密存储。 |
| **H-03** | **知识库文件上传与解析** | 后端/KB | PDF/Excel文件 | 解析后的文本块, 存入Elasticsearch/向量库 | 1. 支持PDF/Excel上传。<br>2. 文件内容被正确解析并存储到ES/向量库中。<br>3. `knowledge_base`表有对应记录。 |
| **H-04** | **RAG检索服务** | 后端/AI | 用户问题 (Query) | 相关的知识片段 (Context) | 输入问题，能从ES/向量库中检索到Top-3相关度高的文档片段。 |
| **H-05** | **智能问答接口 (LLM集成)** | 后端/AI | 用户问题, 上下文, 用户ID | AI回答 (Answer) | 1. 结合检索到的知识片段生成回答。<br>2. 回答流畅且基于知识库内容。 |
| **H-06** | **权限拦截与差异化服务** | 后端/Gateway | JWT Token | 允许/拒绝/功能降级 | 1. 未登录(游客)只能问基础问题(如价格)。<br>2. 登录用户(注册用户)可问详细技术问题。<br>3. 识别到未授权操作返回403。 |
| **H-07** | **咨询日志异步记录** | 后端/Log | 问答对, 用户ID, 耗时 | `consultation_logs` 表新增记录 | 每次对话结束后，数据库中增加一条完整的日志记录(含问题、回复、用户类型)。 |
| **H-08** | **Web端简易聊天窗口** | 前端 | 用户输入 | 聊天气泡界面 | 1. 类似微信的对话界面。<br>2. 能发送文字并展示AI回复。<br>3. 展示“游客”或“用户”身份标识。 |

### 2. 完善体验任务 (优先级：中)
在核心流程跑通后，优化体验和管理能力。

| 任务ID | 任务名称 | 模块 | 输入 | 输出 | 验收标准 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **M-01** | **知识库管理后台** | 前端+后端 | 增删改查操作 | 更新后的知识库 | 管理员可在页面上查看、删除或禁用已上传的文档。 |
| **M-02** | **历史记录同步** | 后端/Order | 用户ID | 历史订单列表 | 模拟同步接口，能获取该用户的模拟订单数据，用于后续个性化回答。 |
| **M-03** | **数据看板基础版** | 前端+后端 | 查询时间范围 | 咨询量统计, 满意度分布 | 管理员可见今日咨询总数、用户满意度饼图。 |

### 3. 扩展任务 (优先级：低)
MVP阶段可暂缓，视进度而定。

| 任务ID | 任务名称 | 模块 | 输入 | 输出 | 验收标准 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **L-01** | **微信服务号集成** | 后端/Channel | 微信回调消息 | 自动回复XML | 在微信公众号发送消息，能收到AI回复。 |

## 二、 关键接口列表 (MVP核心)

以下接口覆盖了核心闭环流程，开发时请严格遵守RESTful规范。

### 1. 认证模块 (Auth)
*   **注册**
    *   `POST /api/auth/register`
    *   **Req:** `{ "username": "zhangsan", "password": "xxx", "phone": "138..." }`
    *   **Res:** `{ "code": 200, "data": { "userId": 101 } }`
*   **登录**
    *   `POST /api/auth/login`
    *   **Req:** `{ "username": "zhangsan", "password": "xxx" }`
    *   **Res:** `{ "code": 200, "data": { "token": "eyJh...", "userType": 1 } }`

### 2. 知识库模块 (Knowledge Base)
*   **上传文档**
    *   `POST /api/kb/upload`
    *   **Req:** `MultipartFile file`, `category="product_manual"`
    *   **Res:** `{ "code": 200, "data": { "docId": 50, "chunkCount": 12 } }`
*   **检索测试 (Debug用)**
    *   `POST /api/kb/search`
    *   **Req:** `{ "query": "产品保修期多久？" }`
    *   **Res:** `{ "code": 200, "data": [ { "content": "保修期为2年...", "score": 0.92 } ] }`

### 3. 智能问答模块 (Chat)
*   **发送消息**
    *   `POST /api/chat/send`
    *   **Header:** `Authorization: Bearer <token>` (可选)
    *   **Req:**
        ```json
        {
          "message": "这款手机多少钱？",
          "history": [ ... ] // 可选，上一轮对话上下文
        }
        ```
    *   **Res:**
        ```json
        {
          "code": 200,
          "data": {
            "reply": "根据最新的报价单，这款手机售价为2999元。",
            "source": "报价单2023.xlsx",
            "usage": { "tokens": 150 }
          }
        }
        ```

### 4. 数据洞察模块 (Analytics)
*   **获取今日概览**
    *   `GET /api/admin/stats/daily`
    *   **Req:** `date="2023-10-27"`
    *   **Res:** `{ "code": 200, "data": { "total_chats": 120, "avg_satisfaction": 4.5 } }`
