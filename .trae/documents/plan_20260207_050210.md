# Implementation Plan for User Registration/Login

## Backend Implementation

### 1. Dependencies (`pom.xml`)

* Added Spring Data JPA, Spring Security, Validation, MySQL Driver, Lombok, JWT (jjwt).

### 2. Entity & Repository

* **Entity**: `User.java` mapped to `users` table.

* **Repository**: `UserRepository.java` extending `JpaRepository`.

### 3. Security & Service

* **JWT**: `JwtUtils.java` for token generation and validation.

* **Service**: `AuthService.java` handling business logic (password hashing, user creation).

* **Config**: `SecurityConfig.java` configuring `BCryptPasswordEncoder` and allowing access to `/api/auth/**`.

### 4. Controller

* **AuthController**: Exposes `/api/auth/register` and `/api/auth/login`.

### 5. Tests

* **AuthControllerTest**: Integration tests using `MockMvc` to verify registration and login flows.

## Frontend Implementation

### 1. Configuration

* **Router**: Configured `vue-router` in `src/router/index.js`.

* **API**: `src/api/auth.js` for Axios calls.

### 2. Views

* **Login.vue**: Login form with token storage.

* **Register.vue**: Registration form.
